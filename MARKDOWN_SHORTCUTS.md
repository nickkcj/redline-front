# Atalhos Markdown - Estilo Obsidian

O editor Tiptap agora suporta todos os atalhos de Markdown do Obsidian! Digite os comandos e pressione **espaço** para ativar.

## Headings (Títulos)

- `# ` → Heading 1 (H1)
- `## ` → Heading 2 (H2)
- `### ` → Heading 3 (H3)
- `#### ` → Heading 4 (H4)
- `##### ` → Heading 5 (H5)
- `###### ` → Heading 6 (H6)

## Formatação de Texto

- `**texto**` ou `__texto__` → **Negrito**
- `*texto*` ou `_texto_` → *Itálico*
- `~~texto~~` → ~~Tachado~~
- `` `código` `` → `Código inline`
- `==texto==` → Destaque (highlight)

## Listas

- `- ` ou `* ` → Lista com marcadores
- `1. ` → Lista numerada
- `- [ ] ` → Checkbox (tarefa não concluída)
- `- [x] ` → Checkbox (tarefa concluída)

## Blocos

- ``` ` ` ` ``` (3 backticks) → Bloco de código
- `> ` → Citação (blockquote)
- `---` ou `***` → Linha horizontal

## Links e Imagens

- `[texto](url)` → Link
- `![alt](url)` → Imagem

## Atalhos de Teclado

### Formatação
- `Ctrl/Cmd + B` → Negrito
- `Ctrl/Cmd + I` → Itálico
- `Ctrl/Cmd + U` → Sublinhado
- `Ctrl/Cmd + Shift + S` → Tachado
- `Ctrl/Cmd + E` → Código inline

### Headings
- `Ctrl/Cmd + Alt + 1` → H1
- `Ctrl/Cmd + Alt + 2` → H2
- `Ctrl/Cmd + Alt + 3` → H3

### Listas
- `Ctrl/Cmd + Shift + 7` → Lista numerada
- `Ctrl/Cmd + Shift + 8` → Lista com marcadores
- `Ctrl/Cmd + Shift + 9` → Checklist

### Outros
- `Ctrl/Cmd + K` → Adicionar link
- `Ctrl/Cmd + Z` → Desfazer
- `Ctrl/Cmd + Shift + Z` → Refazer
- `Ctrl/Cmd + S` → Salvar
- `Shift + Enter` → Quebra de linha
- `Enter` → Novo parágrafo

## Comandos com /

Digite `/` para abrir o menu de comandos:

- `/h1` até `/h6` → Headings
- `/bullet` → Lista com marcadores
- `/number` → Lista numerada
- `/check` → Checklist
- `/quote` → Citação
- `/code` → Bloco de código
- `/table` → Tabela
- `/image` → Inserir imagem

## Menções com @

Digite `@` para mencionar:

- Usuários
- Páginas
- Datas (today, tomorrow)

## Dicas

1. **Markdown automático**: Basta digitar o código e pressionar espaço
2. **Menu contextual**: Selecione o texto para ver opções de formatação
3. **Salvar**: O documento salva automaticamente a cada 2 segundos
4. **Pressione Enter no título**: Move o cursor para o corpo do texto

---

**Nota**: Todos os atalhos funcionam exatamente como no Obsidian!
