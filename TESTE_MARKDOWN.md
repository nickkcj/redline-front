# Teste dos Atalhos Markdown

## Como testar:

1. Abra um documento novo
2. Digite exatamente como mostrado abaixo
3. Pressione **ESPAÇO** após cada comando

---

## Testes para fazer:

### Teste 1: Heading 1
Digite: `# Titulo Grande`
Pressione: ESPAÇO
Resultado esperado: Texto vira H1 (muito grande)

### Teste 2: Heading 2  
Digite: `## Subtitulo`
Pressione: ESPAÇO
Resultado esperado: Texto vira H2 (médio)

### Teste 3: Heading 3
Digite: `### Titulo Menor`
Pressione: ESPAÇO
Resultado esperado: Texto vira H3 (pequeno)

### Teste 4: Negrito
Digite: `**texto negrito**`
Resultado esperado: Automaticamente em negrito

### Teste 5: Itálico
Digite: `*texto italico*`
Resultado esperado: Automaticamente em itálico

### Teste 6: Lista com marcador
Digite: `- item da lista`
Pressione: ESPAÇO
Resultado esperado: Vira lista com bullet point

### Teste 7: Lista numerada
Digite: `1. primeiro item`
Pressione: ESPAÇO
Resultado esperado: Vira lista numerada

### Teste 8: Checkbox
Digite: `- [ ] tarefa`
Pressione: ESPAÇO
Resultado esperado: Vira checkbox desmarcado

### Teste 9: Quote
Digite: `> citacao`
Pressione: ESPAÇO
Resultado esperado: Vira blockquote

### Teste 10: Código inline
Digite: `` `codigo` ``
Resultado esperado: Texto em código
